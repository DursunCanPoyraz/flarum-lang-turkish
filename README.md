# 🇹🇷 Flarum Core Turkish Language Pack

Turkish language extension for Flarum's core.

## Installation

```bash
composer require tolgaaaltas/flarum-lang-turkish
```

## Core list

- 🪐 Core
- ✔ Akismet
- ☑ Approval
- 🕵️ Login with Facebook
- 👨‍💻 Login with GitHub
- 😂 Emoji
- 🏴 Flags
- 👍 Likes
- 🔒Lock
- #️⃣ Markdown
- 💬 Mentions
- 🆕 Pusher
- 👨‍💼 Statistics
- 📌 Sticky
- 🔔 Subscriptions
- 📨 Suspend
- 🏷️ Tags
- 🛑 Spamblock
- ✅ Validation

## Usage

```
Just activate the extension.
```
